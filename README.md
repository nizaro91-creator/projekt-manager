# Benutzerverwaltung (Java Projekt)

Dieses Projekt ist eine einfache Benutzerverwaltung, die im Rahmen meiner Umschulung zum Fachinformatiker für Anwendungsentwicklung erstellt wurde.

##  Projektbeschreibung

Die Anwendung ist ein Konsolenprogramm in Java, mit dem Benutzer verwaltet werden können.  
Das Programm basiert auf objektorientierter Programmierung (OOP) und enthält verschiedene Benutzer-Typen.

##  Verwendete Konzepte

- Objektorientierte Programmierung (OOP)
- Klassen und Objekte
- Vererbung (Benutzer, Admin, Gast)
- Kapselung (private Attribute, Getter/Setter)
- Konstruktoren (überladen und verkettet)
- ArrayList
- Scanner (Benutzereingaben)
- Kontrollstrukturen (if, switch, while)

##  Klassenstruktur

- `Benutzer` (Basisklasse)
- `Admin` (erbt von Benutzer)
- `Gast` (erbt von Benutzer)

##  Funktionen

Das Programm bietet ein Menü mit folgenden Funktionen:

1. Benutzerliste anzeigen  
2. Benutzer hinzufügen (Benutzer, Admin, Gast)  
3. Benutzer löschen (über E-Mail)  
4. Benutzer suchen  
5. Benutzer bearbeiten  
6. Spezialfunktionen:
   - Nur Admins anzeigen  
   - Nur Gäste anzeigen  
   - Mehrere Benutzer löschen  
   - Gastnamen ändern  
0. Programm beenden  

##  Beispiel (Menü)
=============== Benutzerverwaltung ===============

Benutzerliste anzeigen
Benutzer hinzufügen
Benutzer löschen
Benutzer suchen
Benutzer bearbeiten
Spezialfunktionen
Programm beenden



## Ziel des Projekts

Dieses Projekt wurde erstellt, um meine Kenntnisse in Java und objektorientierter Programmierung zu vertiefen und praktische Erfahrung im Umgang mit Benutzerverwaltungssystemen zu sammeln.
